# TecnoShop — Loja

Site público estático para Cloudflare Pages. O arquivo `cliente.html` exibe seleções privadas usando os links gerados no painel.

Antes de publicar, edite `config.js` e informe o endereço do Worker. Esta pasta não contém o painel nem a chave do administrador.

As imagens dos produtos devem ficar em `assets/produtos`. Depois de colocar um arquivo nessa pasta, publique a loja e informe o caminho `/assets/produtos/nome-do-arquivo.webp` no painel.
